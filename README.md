# 【Agent练习作品】驶向电气化的未来 -- 小米SU7 Max 车主分享会 Web PPT

## 使用场景
本作品定义了一个场景：用户要参加一个小米汽车的车友交流会，并且自身要作为演讲者，向车友们讲解自己的爱车（型号为Xiaomi Su7 Max）。分为3个章节——该车性能、使用体验、对生活与家庭的意义。

## 使用方式

1. 下载 `index.html` 和 `assets/` 文件夹，保持目录结构不变
2. 用浏览器（Chrome / Edge / Firefox 均可）打开 `index.html`
3. 按 **F11** 进入全屏模式
4. 使用**鼠标滚轮**上下滚动即可横向翻页
5. 演讲结束后按 **F11** 退出全屏

## 目录结构

```
index.html          # 主文件，包含全部 9 页内容与样式
assets/
  motion.min.js     # WebGL 流体背景动效引擎
  1.png             # 页面配图
  2.png             # 页面配图
  3.png             # 页面配图
```

## 页面概览

共 9 页，分为三个章节：
**第一章：浪潮将至 -- 新能源观瞻** — 封面、数据大字报、智能化观点
**第二章：极致驾控 -- SU7 Max 深度体验** — 章节幕封、配置参数卡、驾驶感受
**第三章：情感共鸣 -- 家庭与生活** — 章节幕封、引用页、家庭意义升华

## 设计风格
- 整体采用"电子杂志 x 电子墨水"美学，深色封面 + 纸白内页
- 主题色为靛蓝瓷（Indigo Porcelain），匹配科技与汽车发布场景
- 中文使用思源宋体，英文搭配衬线体，营造印刷质感
- 背景含 WebGL 流体动效，低性能设备会自动降级

## 注意事项
- 确保 `index.html` 与 `assets/` 文件夹在同一目录下，否则图片和动效无法加载
- 如遇卡顿，WebGL 动效会自动关闭以保障流畅度
- 推荐使用 1920x1080 或更高分辨率的屏幕获得最佳观感
